# PathToSVG
- Rendering of 3D Paths to SVG, performing a viewplane-optimized orthographic 2D projection
- Supports display of localized measurements
- Built on SkiaSharp

[\[nuget.org\]](https://www.nuget.org/packages/EliDll.PathToSVG)
